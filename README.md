# Hotaru
My own game library inspired a lot by raylib build with OpenGL and C. 
By inspired I actually mean that this project is me trying to reimplement 
raylib while exploring it's source code. This is just experimental although 
it's working. In the future, I would like to remake library like this using 
Vulkan.

## What I've learn
- Better event managing system (this is the most important one)
- OpenGL rendering technique
